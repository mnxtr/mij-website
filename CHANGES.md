# Change Log: MIJ Website Redesign

All notable changes to the MIJ Website project are documented here.

## [1.1.0] - 2026-02-22

### Added
- **Theme Engine:** Full dark/light mode support with animated toggle and zero-flash persistence.
- **Premium Design Tokens:** Integrated "Midnight Velvet" (Dark) and "Ivory Paper" (Light) palettes using Tailwind CSS v4.
- **Staggered Animations:** Implemented motion/react reveal variants across all major page entry points.
- **Magnetic UI:** Added responsive magnetic interactions to high-conversion call-to-action buttons.
- **New Typography:** Integrated **Outfit** (Headers) and **Sora** (Body) for a modern architectural aesthetic.
- **E2E Test Suite:** Added Playwright-powered basic specification tests for core user journeys.
- **CI/CD Workflows:** Improved deployment and health-check workflows via GitHub Actions.

### Changed
- **Page Overhauls:** Completely redesigned 12+ pages including Home, About, Services, Partners, News, and Recruit.
- **Global Header/Footer:** Refined with glassmorphism, micro-interactions, and better responsive breakpoints.
- **UI Components:** Updated all atomic components (Buttons, Cards, Badges) to follow the new premium shadow and gradient tokens.
- **Translations:** Synced all localized strings with the new premium terminology and page structures.

### Fixed
- Improved contrast ratios in dark mode for better accessibility.
- Optimized image transitions and loading states.
- Refined contact form validation logic.

## [1.0.0] - 2024-02-21 (Initial Release)
- Initial release with basic functionality and translations.
- Core pages: Home, About, Services, Business, Contact.
